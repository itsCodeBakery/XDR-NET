# XDR-Net Manuscript Revisions

This directory contains the code, outputs, figures, tables, logs, and response material produced while addressing the editor and reviewer comments.

## Structure

- `Reviewer 1/`
- `Reviewer 2/`
- `Reviewer 3/`

Only experiments required to substantiate a reviewer-requested claim should be added. Existing manuscript evidence should be reused when a comment can be resolved through clarification, correction, analysis, or improved reporting.
